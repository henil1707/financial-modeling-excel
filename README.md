# Financial Modeling Project (Excel)

## Overview

This project presents a comprehensive financial model built in Excel, including core financial statements and supporting schedules with forecasting.

## Components

* Profit & Loss (P&L)
* Balance Sheet
* Cash Flow Statement
* Fixed Asset Roll Forward
* Financial Liabilities Schedule
* Equity Schedule

## Forecasting

* Revenue and expense projections
* Asset and liability forecasting
* Integrated financial statement modeling

## Tools Used

* Microsoft Excel

## Key Highlights

* Fully linked 3-statement model
* Supporting schedules for detailed analysis
* Forecast-driven financial insights
